Inicializa la U\+A\+RT.

Se inicializa la U\+A\+RT para usar una trama de 8 bits de datos sin paridad y con un bit de stop. El módulo usa interrupciones tanto para la recepción como para la transmisión. La comunicación con las rutinas de interrupción se realiza mediante dos colas.
